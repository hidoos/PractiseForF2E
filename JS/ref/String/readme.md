# js字符串处理

## api有哪些？


## 需求：

将一个字符串转换成为加空格的字符串，如下：

    var str1 = "hello,world";
    specify(str1); // => "h e l l o , w o r l d"

在字符串字面量上使用方法，如下：

    'hello,world".specify();

有一个字符串：abcd-ef-ghi，请用JavaScript将它处理成：ghi&ef&abcd，练习题目来自 [08年出的一些前端面试题][f2etest]
